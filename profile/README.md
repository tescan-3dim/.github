# TESCAN 3DIM

## Organizational Update

As of **January 1**, TESCAN 3DIM has officially merged with **TESCAN GROUP**.

As part of this integration, all repositories, projects, and ongoing development activities have been consolidated under the official GitHub organization:

👉🏻 https://github.com/tescangroup 👈🏻

## What This Means

- This TESCAN 3DIM organization is no longer actively maintained.
- Active development continues within **TESCAN-GROUP**.
- Issues, pull requests, and new contributions should be directed to the respective repositories under the TESCAN GROUP organization.

---

## For Contributors

If you have existing forks or local clones, please update your Git remotes accordingly:

```bash
git remote set-url origin git@github.com:tescangroup/<repository-name>.git
```
